BASICOS* *cd (Sobre o comando cd Esse comando basicamente permite o usuário alterar o diretório de trabalho.) *ls (O comando ls do Linux O comando ls é usado para listar arquivos ou diretórios no Linux e em outros sistemas operacionais) *ls -lha ( vai listar todos os tamanhos de arquivos em formatos fáceis, como MB, GB ou TB.) *cp(Use o comando cp para copiar arquivos ou diretórios e seu conteúdo. Abaixo, listamos alguns exemplos. Para copiar um arquivo do diretório atual para outro, digite cp seguido do nome do arquivo e do diretório de destino. Por exemplo: cp nomedoarquivo.txt /home/username/Documents) *rm (O comando rm é usado para excluir arquivos em um diretório.) *rm -rf (-rf permite que o sistema faça a remoção sem confirmação.) *mv(O uso mais comum do comando mv é mover arquivos, mas ele também pode ser usado para renomear arquivos. Basta digitar mv seguido do nome do arquivo e do diretório de destino. Por exemplo, você deseja mover o arquivo nomedoarquivo.txt para o diretório /home/username/Documents: mv nomedoarquivo.txt /home/username/Documents. Você também pode usar o comando mv para renomear um arquivo: mv nomedoarquivo_antigo.txt nome_novo.txt) *mkdir -p(o comando mkdir para criar um ou vários diretórios de uma só vez e definir permissões para cada um deles. (-p ou –parents cria um diretório entre duas pastas existentes. Por exemplo, mkdir -p Music/2020/Songs criará o novo diretório “2020”.)) *apt-get(O apt-get é uma ferramenta de linha de comando para lidar com as bibliotecas da Advanced Package Tool (APT) no Linux.A execução do comando apt-get exige que você tenha privilégios sudo ou root.) *cat( Ele é usado para visualizar, criar e relacionar arquivos.Por exemplo: cat nomedoarquivo.txt.) *pwd(para encontrar o caminho para o diretório atual (da pasta) em que você está.) *sudo(permite executar tarefas que exigem permissões administrativas ou de root.) *grep(Ele procura por padrões especificados pelo usuário dentro de arquivos de texto) * | (pipe permite usar dois ou mais comandos, de forma que a saída de um comando sirva como entrada para o próximo.)
